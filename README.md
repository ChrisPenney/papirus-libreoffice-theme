# Install
**Archlinux (AUR)**:
```
yaourt -S papirus-libreoffice-theme
```
**Ubuntu 14.04/16.04**:
```
sudo add-apt-repository ppa:varlesh-l/papirus-pack
sudo apt-get update
sudo apt-get install libreoffice-style-papirus 
```
